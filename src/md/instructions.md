# EFT Skill Instruction Index and Validation Audit

Primary benchmark: `src/md/EFT_Exercises_Extracted.md`, extracted from Rhonda Goldman's deliberate-practice EFT exercises. The benchmark covers Exercises 1-12. The app also contains an experimental local extension, Empathic Refocusing, which is not part of the Goldman benchmark.

## Validation Standard

Use the Goldman examples as the gold standard for:

- what kind of client statement functions as a marker for each skill;
- what the suggested therapist response should demonstrate;
- how difficulty increases from beginner to intermediate to advanced;
- where a skill boundary differs from neighboring skills.

For app content, stable item IDs and the case/skill matrix can stay fixed, but each item should be judged by whether the client statement naturally invites the target skill and whether the suggested response stays skill-pure.

## Skill Boundaries

- Empathic understanding reflects what is already explicit.
- Validation says why the feeling is legitimate in context.
- Exploratory questions ask the client to sense into unclear experience.
- Empathic explorations stay with the current live edge and invite a little more.
- Empathic evocations provide vivid sensory or imagistic language.
- Empathic conjectures tentatively name a near-surface feeling not yet owned.
- Treatment rationale explains why the emotional method or task helps.
- Staying in contact holds intense affect safely without fleeing or flooding.
- Self-disclosure uses brief therapist transparency for the client's benefit.
- Chair-work setup recognizes a task marker and launches the correct enactment.
- Alliance repair works directly with rupture in the therapy relationship.
- Empathic refocusing, as a local extension, names avoidance and invites return to the just-left feeling.

## Content QA Sequence

1. Check the marker first: does this client statement naturally invite the target skill?
2. Check the response second: does it demonstrate only that skill, without stacking tasks?
3. Check difficulty: does the hard version add relational complexity, risk, defense, or ambiguity?
4. Check case voice: does the statement sound like the assigned case, not a generic skill prompt?
5. Check safety: is high-risk material held clinically without panic, avoidance, or overreach?
6. Check translation last: Norwegian should localize the final English clinical intent in natural Bokmal.
